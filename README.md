# Dark Minimal Calculator (HTML / CSS / JS)

A compact, mobile-friendly calculator UI built using pure HTML, CSS, and JavaScript.  
Includes basic arithmetic operations, percentage, toggle sign, clear, and backspace.  
Designed with a sleek dark theme optimized for phone browsers.

---

## Features

- Clean dark UI with rounded elements  
- Classic calculator layout 
- Responsive layout suitable for all mobile screens 

---

## How to Run

1. Open the Link -> .
2. Open it in any modern browser on phone or desktop.

---

## Notes

- The calculator uses JavaScript’s built-in expression evaluator.  
- No external libraries or frameworks are required.
